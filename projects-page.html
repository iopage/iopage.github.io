Multiplication Table Generator

This is made from javascript, when you type your input as an integer and click print, the table of that number get printed. However with decimal numbers it can cause mistakes in printing due to floating point errors.

How it works: When you type any integer in the input, it gets stored in a variable.
When you click "print" a javascript function works which add text in pre existing div in the form of n × i = ni
This step is repeated for values of i from 1 to 10 with the help of 'for loop'.

Loop function in javascript works like sigma function in maths. The only difference is that sigma (∑) do summation and 'for loop' do whatever we want (like printing).
